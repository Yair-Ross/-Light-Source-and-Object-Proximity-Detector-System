MSP Side:
	main.c - Responsible for the FSM logic.
	api.c - Responsible for the application related functions.
	hal.c - Responsible for the hardware related functions.
	bsp.c - Responsible for the hardware configurations.
	The files' headers include the function and variable declarations.

PC Side: (Developed using Visual Studio 2022)
	MainWindow.xaml - Responsible for the UI design configurations.
	MainWindow.xaml.cs - Responsible for the UART communication and UI updates.
	Atia.cs - Auxiliary class for UI display of sonar marks.
